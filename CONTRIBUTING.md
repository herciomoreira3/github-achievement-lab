# Contributing

Terima kasih sudah tertarik berkontribusi.

## Workflow

1. Buka issue untuk menjelaskan perubahan.
2. Buat branch dari `main`.
3. Commit perubahan kecil yang mudah diverifikasi.
4. Buat pull request dengan ringkasan yang jelas.
5. Tunggu hasil pengecekan sebelum merge.

## Contribution Style

- Jaga perubahan tetap fokus.
- Hindari pull request kosong atau perubahan yang tidak bermakna.
- Pakai bahasa yang jelas di issue dan pull request.
